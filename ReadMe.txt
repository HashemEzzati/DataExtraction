In this code, the famous candlestick patterns of the financial data of 
several Iranian companies are extracted using the method introduced in
(Hu, W., Si, Y.W., Fong, S. and Lau, R.Y.K., 2019. A formal approach to
 candlestick pattern classification in financial time series. 
 Applied Soft Computing, 84, p.105700.)


Once a pattern is detected, the pattern with 9 candles befor that is 
saved Since we want to apply this data for our purpose according to 
the method in (Chen, J.H. and Tsai, Y.C., 2020. Encoding candlesticks 
as images for pattern classification using convolutional neural 
networks. Financial Innovation, 6(1), pp.1-19.)
